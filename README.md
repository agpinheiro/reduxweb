# reduxweb

Exemplo de Carrinho usando REACT E REDUX

git clone https://github.com/agpinheiro/reduxweb.git

Requisitos:

npm, React, yarn

iniciar o projeto:

yarn //para gerenciar dependencias

yarn start // rodar o projeto
